# ADR-0012: argon2id 비밀번호 해싱 전환 (Lazy Migration)

- Status: Accepted
- Date: 2026-05-12
- Phase: Phase 5 (TP8)
- Tags: security, password-hashing, migration

## Context and Problem Statement

현 구현은 `SHA256 3회 반복` (Causes of Redesign + Smell: Oddball Solution + AntiPattern: Reinvent the Wheel — problem.md TP8 근거). 보안 학습 가치 측면에서 업계 표준 알고리즘으로 전환 필요. 단, 기존 사용자 비밀번호를 한 번에 재해싱할 수단이 없다 (평문 보유 안 함).

Forces:
- TP8 / BP6 프로덕션 품질 개선 학습 목표
- OWASP Password Storage Cheat Sheet 2024 1순위 권장 = argon2id
- 기존 데이터 보존 — Big-Bang 일괄 마이그레이션 불가 (평문 없음)
- Phase 4 부하 테스트 기준점 보호 — 측정 중 변경 금지

## Considered Options

(1) argon2id + Lazy Migration (선택)
- 알고리즘: argon2id (Argon2 PHC 2015 우승, memory-hard, GPU/ASIC 저항)
- 파라미터 (OWASP 2024 최소): memoryCost=19456 (19 MiB), timeCost=2, parallelism=1
- 마이그레이션: 로그인 성공 시점에 재해싱 → argon2id로 교체 (PHC string prefix `$argon2id$`로 판별)
- Pros: 최신 표준, GPU 저항, Lazy Migration으로 무중단, 기존 비밀번호 알고리즘 prefix로 자연 판별
- Cons: 모든 사용자가 1회 로그인할 때까지 구식 해시 잔존

(2) bcrypt
- Pros: 검증 이력 가장 길어 안정적
- Cons: GPU 공격 상대적 취약, OWASP 2순위, 학습 가치 측면에서 최신 표준(argon2id) 우위

(3) PBKDF2
- Pros: NIST 권장
- Cons: 구식, memory-hard 아님

(4) Big-Bang 일괄 마이그레이션
- Pros: 즉시 완전 전환
- Cons: 평문 미보유로 불가능 — 강제 비밀번호 재설정 필요 (UX 파괴)

(5) SHA256 3회 반복 유지
- Pros: 변경 없음
- Cons: Smell + AntiPattern 지속, BP6 학습 목표 미달성

## Decision Outcome

(1) 채택. Phase 5 진입 시 전환.

라이브러리: `argon2` (Node.js 네이티브 바인딩).

Expand-and-Contract + Lazy Migration:
- Phase 5 진입 직후: `user_auth.password` 컬럼 스키마 동일 (VARCHAR(1000)), 값 포맷만 변경
- 검증 로직: `$argon2id$` prefix → argon2.verify(), 아니면 기존 SHA256 검증
- 로그인 성공 시점: SHA256 검증 통과한 비밀번호를 평문 그대로 argon2id로 재해싱 후 DB UPDATE
- 모니터링: SHA256 잔존 비율 (`password NOT LIKE '$argon2id$%'` 카운트). 임계값 도달 시 잔존 사용자 대상 비밀번호 재설정 안내

## Consequences

긍정:
- OWASP 2024 최신 권장 알고리즘 운용 경험 (BP6)
- GPU/ASIC 공격 저항 (memory-hard)
- 사용자 무중단 (Lazy Migration)
- STRIDE-1 (Spoofing: 로그인 brute-force) 완화 강화

부정:
- Phase 5 의존성 추가 (`argon2` 네이티브 빌드 필요 — Docker 빌드 시간 영향)
- 1회 로그인 안 한 사용자는 SHA256 잔존 — 영구 잔존 가능성 (강제 만료 정책 없으면)
- Phase 4 baseline 대비 응답 시간 증가 예상 (argon2id memoryCost 19 MiB) — Phase 5 재측정의 명시적 변수
